# RAP2RUE

Plateforme MERN dédiée à la musique rap pour artistes et fans.

## Fonctionnalités principales
- Inscription/connexion artistes et fans (JWT)
- Upload de musiques, EPs, albums (Cloudinary)
- Parcours, extraits gratuits, achat et streaming sécurisé (Paydunya)
- Tableau de bord artistes (ventes, écoutes)
- Avis/notes sur projets achetés
- UI noir & blanc, responsive, inspirée de Spotify

## Stack technique
- **Frontend** : React (Vite), Redux Toolkit, React Router, TailwindCSS
- **Backend** : Node.js, Express, MongoDB, JWT, bcrypt, Cloudinary, Paydunya

## Lancement rapide
1. `npm install` (frontend)
2. `cd backend && npm install` (backend)
3. Configurer les variables d'environnement (voir `.env.example`)
4. Lancer le frontend : `npm run dev`
5. Lancer le backend : `npm start`

---

> Ce projet vise à soutenir les artistes rappeurs en leur permettant de monétiser leur musique via une plateforme dédiée.
